# Manutenção

## Atualizações
- Verifique regularmente as atualizações disponíveis no repositório oficial.
- Para atualizar, baixe a nova versão e substitua os arquivos existentes, exceto `config.php`.
- Execute o script de atualização acessando `http://seusite.com/update`.

## Solução de Problemas
### Erro de Conexão com o Banco de Dados
- Verifique as credenciais no arquivo `config.php`.
- Certifique-se de que o servidor MySQL está em execução.

### Página Não Carrega Corretamente
- Limpe o cache do navegador.
- Verifique o log do servidor para identificar erros.
